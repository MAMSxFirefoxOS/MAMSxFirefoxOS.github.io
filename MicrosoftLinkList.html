MAMS x Firefox OS

MBaaSとは?

Microsoft Azure の MBaaS ( Mobile Backend as a Service )は、スマートフォンアプリ開発で必要となる汎用的なAPI集です。サーバ側のコードを書く事無く、手早くアプリケーション制作が行えます。Android / iOS / WindowsPhone などの開発では、MBaaSを有効活用する事で、効率的に開発が可能です。
しかし、 Firefox OS では、様々な点で対応していない部分があります。プッシュ通知などがそうです。当サービスを使えば、Microsoft Azure x Firefox OS で (ある程度は)MBaaS が使用できるようになります。
ソースコードは GitHub 上にオープンソースで公開されています。
Firefox OS でアプリケーションを開発する事が、ちょっと楽で、楽しくなりかも。
